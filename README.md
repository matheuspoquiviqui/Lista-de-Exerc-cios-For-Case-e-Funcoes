# 🐍 Lista de Exercícios — Estruturas de Repetição e Funções em Python

Este repositório contém uma lista de 10 exercícios desenvolvidos em Python com foco em estruturas de repetição (`for` e `while`), tomada de decisão (`if`, `elif`, `else`), estrutura `match case` e criação de funções.

## 🎯 Objetivo

Praticar conceitos fundamentais da programação em Python, desenvolvendo habilidades de lógica, repetição de processos, controle de fluxo e reutilização de código através de funções.

---

## 📂 Estrutura do Projeto

```bash
exercicios/
├── exer01.py
├── exer02.py
├── ...
└── exer10.py
```

Cada arquivo representa uma atividade individual.

---

## ▶️ Como executar os exercícios

1. Certifique-se de ter o Python instalado em seu computador.
2. Abra o terminal na pasta do projeto.
3. Execute o exercício desejado:

```bash
python exercicio01.py
```
